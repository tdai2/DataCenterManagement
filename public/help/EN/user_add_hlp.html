Adding User - Help

Use this page to add a new user. The new user will be automatically enabled for LAN access.

User Name

Enter the name of the new user.

  • - User Name is a string of 1 to 16 alpha-numeric characters or '_' or '-'.
  • - User Name must begin with an alphabetic character.

The names below are reserved and may not be used as a User Name.

  • - anonymous
  • - daemon
  • - games
  • - halt
  • - mail
  • - news
  • - nobody
  • - operator
  • - root
  • - shutdown
  • - sshd
  • - stunnel4
  • - sync
  • - uucp

Password, Confirm Password

Enter and confirm the new password here. The password must be between 0 and 20 characters long.

Network Privilege

The level of network privilege to assign to this user.

  • - Administrator
  • - Operator
  • - User
  • - Callback
  • - No Access
Click here to get detailed network privilege of each level
Actions
Add

Click the 'Add' button to save the new user and return to the user list.

Cancel

Click on cancel to return to the list without adding the new user.
