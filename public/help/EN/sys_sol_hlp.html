System SOL Log - Help

Use this page to collect system SOL information about the server.

System SOL Log

The System SOL Log data is used by the system manufacturer for analysis.

Latest Log

Shows the time of the latest data collection.

Generate Log

Click the Generate Log button to collect recent SOL Log data. The resulting compressed archive will be downloaded to your system by clicking on the sol log link. You may also choose to download the data at a later time using the sol log link. Note that it is recommended that fresh data always be downloaded for analysis.
