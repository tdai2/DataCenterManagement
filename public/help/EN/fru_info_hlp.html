FRU Information - Help

Use this page to view the BMC FRU file data. If a FRU data device is inaccessible, it may be offline due to the power state. Please be sure the system is 'on', then re-load this page using the "FRU Information" link in the navigation panel to refresh the data.

Use the dropdown selector to choose FRU data to view. Only FRU devices with a valid type 11 FRU device locator record in the SDR will be displayed here.
