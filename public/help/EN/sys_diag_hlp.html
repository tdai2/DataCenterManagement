System Debug Log - Help

Use this page to collect system debug information about the server.

System Debug Log

The System Debug Log data is used by the system manufacturer for analysis.

Last Log

Shows the time of the last data collection. Collection times older than three minutes will be marked as an "Old" debug log.

Encryption

The resulting archive will contain two smaller zip files. These smaller zip files will be encrypted for privacy, and may only be extracted for analysis by an authorized representative of the system manufacturer.

  • SystemDebugLog.zip - Contains Baseboard Management Controller (BMC) status, BMC configuration settings, BMC Sensor readings, Power supply data, System Event Log, sensor readings, SMBIOS tables, CPU machine check registers and PCI configuration space information.
  • SystemBMCDebugLog.zip - Contains regular System Debug Log plus the BMC debug log.

Generate Log

Click the Generate Log button to collect recent Debug Log data. The resulting compressed archive will be downloaded to your system by clicking on the debug log link. You may also choose to download the data at a later time using the debug log link. Note that it is recommended that fresh data always be downloaded for analysis.
