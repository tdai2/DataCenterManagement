Serial Over LAN - Help

Serial Over LAN

Enable or disable Serial Over LAN for each LAN channel.

If LAN failover is enabled, invalid channels will be disabled.

Save

Click 'Save' to enable SOL settings.

SMASH Port

Allows changing the port used by the SMASH (Systems Management Architecture for Server Hardware) interface. Changing this setting will immediately terminate all open SMASH sessions.

Actions
Save

Click 'Save' to set the SMASH port.
