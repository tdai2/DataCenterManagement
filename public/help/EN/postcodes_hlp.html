System POST Codes - Help

Use this page to view recent Power-On Self Test results.

The timebase may be viewed as the time from start of POST, or time since the previous POST code was logged. All time formats are in minutes:seconds.milliseconds.

Previous and current boot POST codes are shown. The current boot codes become previous codes when the system is reset or shut down.

Holding the cursor over a time, POST code, or description will highlight all other occurrences of that same POST code. Clicking on a time, POST code, or description will cause highlighting to persist until another code is clicked.
