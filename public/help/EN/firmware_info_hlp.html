Firmware Information - Help

Use this page to view the firmware version numbers for this server.

BMC FW Rev

Major and minor revision of the BMC firmware.

BMC FW Build Time

The date and time of the installed BMC firmware.

Boot FW Rev

Major and minor revision of the BOOT firmware.

Primary HSC FW Rev

Major and minor firmware revision for the Primary HSC (Hot Swap Controller; also known as HSBP, Hot Swap Backplane). Only available if the host is powered on.

Secondary HSC FW Rev

Major and minor firmware revision for the Secondary HSC (Hot Swap Controller; also known as HSBP, Hot Swap Backplane). Only available if the host is powered on.

ME FW Rev

Major and minor firmware revision for the Management Engine (ME). Only available if the host is powered on.

LCP FW Rev

Major and minor firmware revision for the Local Control Panel (LCP). Only available if the host is powered on.
