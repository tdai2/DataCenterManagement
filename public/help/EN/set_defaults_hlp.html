Set BMC Defaults - Help

Use this page to reset all BMC settings to factory defaults.

This operation cannot be undone.

Caution: This operation cannot be undone. Once complete, all remote management, including the web server, will not be accessible until users and network settings are restored locally.

Settings lost include, but are not limited to:

  • All network addresses and settings
  • Power restore policies
  • Platform event filters
  • Alert destinations

This will not affect the BMC's system event log, sensor data repository, or any Node Manager settings and policies.
