Modify User - Help

Use this page to modify a user's password and permissions.

User Name

The name of the user being configured (READ ONLY).

Change Password

To change the user's password, check this box which will enable the password fields.

Password, Confirm Password

Enter and confirm the new password here. The password must be between 0 and 20 characters long.

Enable Access

Check this box to enable the user for LAN access. Also enables/disables SOL, KVM, Media Redirection.

Network Privilege

The level of network privilege to assign to this user.

  • - Administrator
  • - Operator
  • - User
  • - Callback
  • - No Access
Click here to get detailed network privilege of each level
Actions
Modify

Click 'Modify' to accept the modifications and return to the user list.

Cancel

Click 'Cancel' to return to the list without modifying the new user.
